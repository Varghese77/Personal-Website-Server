README!

          This app is a simple chatroom web application where users enter the chatroom anonymously and can then specify a custom handle. Users can also create custom rooms to chat in and all messages are encrypted via HTTPS and are not logged on the server (note that anyone can enter a room though).
          

          

          To use the VOIP functionality click the downward arrow button. Enter the handle of someone who you wish to call and press the green button to initiate a call offer. If the other user accepts then the application will try to exchange information needed to set up a peer-to-peer connection using WebRTC. Calls will auto-decline after 10 seconds and you can cancel a call at anytime with the red hangup button.
          

          

          NOTE: VOIP functionality only works on the latest verzions of Chrome and Firefox.
