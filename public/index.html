Notable Coursework

AMATH 301: Beginning Scientific Computing

In this class I learned the basics of computational mathematics and how efficiently calculate answers to complex numerical problems including how to use various integration schemes to predict chaotic systems, how to use the fourier transform to remove noise from an audio signal, and even how to use eigen vectors/values to build a facial recognition algorithm.

CSE 351: Hardware/Software Interface

The point of this class is to teach students a sense of "what really happens" under the hood when software runs. Topics explored key computational abstraction ideas below modern high-level languages such as number representation, assembly language, introduction to C, memory management, the operating-system process model, high-level machine architecture including the memory hierarchy, and how high-level languages are implemented

CSE 333: Systems Programming

This class taught me advanced knowledge of the C and C++ programming languages as well as various topics relevent to creating programs on a system level including memory management, networking and various methods of concurrent programming. The final project of this class was to build a C document index hosted on a server built entirelly with C++.

CSE331: Software Design and Implementation

This class explored concepts and techniques for the design and construction of reliable and maintainable software systems in modern high-level languages such as specifications, program structure and design, program correctness approaches, testing, and event-driven programming (e.g., graphical user interface)."

CSE 312: Foundations Of Computing II

This class thought me the fundamentals of statistics with regards to computation including counting, various statistical distributions, the Central Limit Theorm, polynomial-time versus NP; and NP-completeness, and Bayes Theorem.

CSE 332: Data Structures and Parallelism

This class covers abstract data types and structures including dictionaries, balanced trees, hash tables, priority queues, and graphs; sorting; asymptotic analysis; fundamental graph algorithms including graph search, shortest path, and minimum spanning trees; concurrency and synchronization; and parallelism.
